### linear-congruential-generators
Linear Congruential Generators
Time Complexity: O(n)

### Question
Examine the following random number generator:
Xn+1=(A×Xn+B)%M A, B, X1, K and M will be given. 
Output the random numbers from K-th to K+4-th.

### Sample Input 1
3 2 5 1 7

### Sample Output 1
5
3
4
0
2


### Sample Input 2
1234 5678 123 12345 9876

### Sample Output 2
4162
6066
5114
5590
414
